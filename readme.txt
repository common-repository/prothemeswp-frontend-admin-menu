== Description ==

ProThemesWP Frontend Admin Menu displays WordPress's admin menu on the frontend of your website to make accessing WordPress's settings pages quick and easy.

The admin menu is sticky and can be accessed even after scrolling down.

The admin menu is only displayed to logged in users that use admin pages.

If you have any questions, please don't hesitate to create a new topic on the <a href="https://wordpress.org/support/plugin/prothemeswp-frontend-admin-menu">WordPress forum</a>.

== Installation ==

Just install and activate ProThemes Frontend Admin Menu on your plugins page.

The admin menu will then appear on the frontend of your website.

The admin menu will be enabled for other users of your website only if they visit admin pages.
